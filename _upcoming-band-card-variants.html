Phase 37 — Upcoming Band Card

Three design directions for a dismissible, expandable card appearing 15 min before next picked band starts on /now. Answers: Expands in place (1A) · Prominently highlighted (2C) · Subtle quack strip (3B).

V1 — Alert Stripe

Thick accent left edge

Key insight: Interrupt attention via 6px hot-red left border + compact density. Expanded state stacks crew list below, keeping visual footprint tight. Quack strip is subtle action footer.

Why this direction

  • Visual hierarchy: 6px accent stripe is immediately scannable, not using color-alone for accessibility.
  • Prominence: Red stripe + tight density = catches eye without elaborate surrounding.
  • Scalability: Expands vertically, doesn't break mobile layout.
  • Quack: Subtle footer button, doesn't compete with crew list.
V2 — Spotlight

Elevated + gentle glow

Why this direction

  • Prominence through atmosphere: Subtle shadow + red glow create visual elevation without being jarring.

V3 — Minimal Notify

Compact, high-contrast alert

Key insight: Ultra-compact collapsed (one-liner with badges), 4px left accent border. Minimal footprint + maximal signal clarity. Expanded stacks crew with icons. Quack is secondary action below list.

Why this direction
