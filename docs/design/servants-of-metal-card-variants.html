Servants of Metal

Design variants for godlike user-management rows. Goal: make Manager, Friend, and Blocked states readable at a glance — without four same-weight buttons fighting for attention.

Open this file in a browser (Chrome/Safari), not the editor markdown preview — the phone mockups need HTML + CSS.

Assumptions (confirm before implementing in React)
  • Actions stay: assign patch, toggle manager, toggle friend, unblock (when blocked).
  • Godlike rows are read-only for role — only shown in samples as non-interactive.
  • Mobile-first (~390px); ~20 users max — scroll is OK, cognitive load is not.
  • Current bug: colored button = «click to turn ON», gray button = «already ON» — inverted from common toggle mental model.
