Notas de design
Tokens usados
Apenas tokens existentes: --bg-elevated, --border, --text, --text-muted, --text-faint, --signal-ok, --font-display, --font-mono, --s-*, --r-*

Sem --signal-danger (não existe) — chips negativos usam --accent (#c0392b / #e74c3c).
Logo MoshSplit
img src aponta para split.viralatas.org/moshsplit/assets/logo.svg. onerror mostra SVG fallback inline (ícone de divisão ✗). Mockup usa fallback porque URL retorna 500 em maio/2026.
Chip de balanço
Negativo: fundo rgba(accent, 0.20) + borda rgba(accent, 0.40) + texto #e74c3c
Positivo: fundo rgba(signal-ok, 0.20) + borda + texto #1abc9c
Settled: versão atenuada do verde, texto "Quitado"
Padrão Collapsible
Exatamente o mesmo padrão de ConflictSection: Collapsible de src/ui, trigger como ReactNode, defaultOpen=false, classe de wrapper via className prop, margin var(--s-3) var(--s-4).
